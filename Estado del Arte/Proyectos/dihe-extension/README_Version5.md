# Dihë

Minimalista extensión para Chrome que te permite acceder a un chat AI (tipo ChatGPT), integrado con GitHub para desarrollo asistido y almacenamiento.

## Instalación paso a paso

1. Descarga estos archivos y extrae la carpeta.
2. Ve a `chrome://extensions` y activa "Modo desarrollador".
3. Haz clic en "Cargar descomprimida" y selecciona la carpeta del proyecto.
4. El icono aparecerá en tu barra de Chrome. Haz clic para abrir el sidebar o usa el atajo:  
   - Mac: `Command + Shift + Y`
   - Windows/Linux: `Ctrl + Shift + Y`
5. Ingresa tu clave de OpenAI en la sección de configuración y haz clic en "Guardar".

## FAQ

**¿Dónde se guarda mi clave de OpenAI?**  
Solo localmente en tu navegador, nunca se envía a servidores externos.

**¿Puedo borrar mi clave?**  
Sí, solo borra el campo y haz clic en "Guardar".

**¿Es accesible?**  
Sí, puedes navegar y usar todo con el teclado.

**¿Qué pasa si la clave es inválida?**  
Aparece un mensaje en rojo.

## Accesibilidad
- Navegación por teclado
- Contraste alto
- Roles y etiquetas ARIA
