Terraza es la aplicación Next.js para captura, codificación GT, grafo y cola de commits del corpus. Por seguridad no se despliega en GitHub Pages — solo corre en entorno local o servidor privado.

Desarrollo local: clona el repositorio, instala dependencias en web/terraza/ y ejecuta npm run dev. Requiere passkey o sesión de investigadora. La URL /terraza/ en GitHub Pages muestra esta página informativa, no la app.
