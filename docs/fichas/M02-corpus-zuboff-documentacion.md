---

## Nota de método

El corpus Zuboff registra citas extraídas del libro con protocolo **Clave B** (lectura de páginas físicas anotadas). Cada entrada en JSON incluye texto citado, categoría temática, color de marcador y notas de aplicación al contra-archivo (mistranslation, vigilancia institucional, AFP/SURA).

Esta ficha no reproduce la teoría de Zuboff: **documenta el archivo de citas** como insumo metodológico del motor de fricción.

## Relevancia analítica

- Capitalismo de vigilancia como marco para leer opacidad previsional y roles bancarios digitales.
- Poder instrumentario y extracción conductual — trasladado a regímenes institucionales chilenos.
- Conexión con poemario/ficha C04 (vigilancia corporativa) y caso SURA.
