# Fichas del Contra-Archivo

Este directorio contiene las fichas de investigación del proyecto Contra-Archivo. El sistema de fichas es el núcleo metodológico del trabajo doctoral: cada ficha codifica un concepto, un sistema o un caso etnográfico siguiendo Grounded Theory (open → axial → selective coding).

## Estructura y convenciones

### Nomenclatura
- `C01-slug.md`, `C02-slug.md`, etc.
- El número correlativo indica orden de producción, no jerarquía analítica
- El slug describe el tema central de forma breve

### Tipos de ficha

1. **Fichas conceptuales**: Desarrollan un concepto teórico o marco analítico
   - Ejemplo: C01 — El núcleo y el margen (concentración infraestructural de datos)

2. **Fichas de sistema**: Analizan un sistema o dispositivo institucional
   - Ejemplo: C02 — La cuenta y el núcleo (sistema de pensiones chileno)

3. **Fichas metodológicas**: Protocolos operativos, definiciones de proceso, gates de control
   - Ejemplo: C03 — Protocolo de buenas prácticas de documentación (clasificación P/R/C, gate de pre-publicación)

4. **Fichas de caso etnográfico**: Documentan casos empíricos específicos
   - [Pendientes]

### Estructura obligatoria de cada ficha

Toda ficha debe incluir:

1. **Encabezado con metadata**
   - Tipo (conceptual / sistema / caso)
   - Línea de investigación
   - Estado (borrador / revisión / publicable)
   - Fuentes (tipo de material utilizado)

2. **Nota de método y alcance**
   - Delimita si el contenido es hecho documentado, hipótesis investigativa o inferencia teórica
   - Declara qué fuentes utiliza y qué NO documenta

3. **Concepto**
   - Formulación del problema o fenómeno central

4. **Lectura teórica**
   - Anclajes en autores y marcos teóricos (Abrams, Stoler, Foucault, Das y Poole, etc.)
   - Cada referencia debe citarse con formato APA

5. **Codificación (Grounded Theory)**
   - **Open coding**: códigos emergentes del material
   - **Axial coding**: relaciones entre códigos
   - **Selective coding**: categoría central que integra el análisis

6. **Dimensiones D1–D7**
   - Mapeo dimensional según esquema del contra-archivo
   - D1: Jurisdicción/soberanía
   - D2: Localización infraestructural
   - D3: Atribución de acceso
   - D4: Régimen archivístico
   - D5: Clasificación del sujeto
   - D6: Fricción inter-capas
   - D7: Visibilidad/opacidad

7. **Sitio de fricción / mistraducción**
   - Identifica el punto donde las tres capas (ética, institucional, material) entran en tensión
   - La fricción NO se resuelve; se expone y se vuelve auditable

8. **Apertura analítica**
   - Preguntas que la ficha deja abiertas
   - Articulaciones pendientes con otras líneas o fichas
   - NO se cierran conclusiones; se documenta el siguiente paso analítico

9. **Vínculos con otras fichas**
   - Enlaces explícitos a fichas relacionadas
   - Breve descripción de la relación

10. **Referencias**
    - Bibliografía académica (formato APA)
    - Registro normativo y documental (leyes, decretos, reportes oficiales)
    - Marcar `[CITA PENDIENTE]` cuando falta verificación

## Reglas duras de contenido

1. **Toda afirmación se ancla en fuente verificable**
   - Documento legal (Ley/DS con número y fecha)
   - Literatura académica (autor, año, paginación)
   - Registro público (reportes, sentencias, movilizaciones documentadas)

2. **Distinguir siempre:** 
   - *Hecho documentado*: afirmación respaldada por fuente pública
   - *Hipótesis investigativa*: interpretación fundada pero no cerrada
   - *Inferencia teórica*: extensión analítica desde el marco teórico

3. **No saneamiento**: Mantener el grano polémico cuando el dato lo sostiene

4. **Privacy by design**: No publicar material que comprometa fuentes vivas, no nombrar sujetos identificables sin protocolo de protección

## Uso interno

Este directorio es de trabajo doctoral. Las fichas en estado "borrador" no están destinadas a publicación en el sitio vivo (`index.html`, `buscador.html`) hasta completar revisión.

La integración al grafo d3-force del sitio público se decide ficha por ficha, evaluando si el material:
1. Proviene exclusivamente de fuentes públicas verificables
2. No compromete protocolos éticos de investigación
3. Aporta al instrumento crítico sin cerrar prematuramente el análisis
